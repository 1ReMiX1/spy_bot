# \U0001F310 Онлайн-режим (Firebase) — настройка за 5 минут

Чтобы игры работали «каждый со своего телефона», нужна бесплатная общая база
данных. Используем **Firebase Realtime Database** (бесплатный тариф Spark).
Свой сервер поднимать НЕ нужно — сайт подключается к базе напрямую.

Без этой настройки кнопка «Онлайн с друзьями» покажет сообщение «Онлайн ещё не
настроен», но режим «С одного устройства» работает сразу.

## Шаг 1. Создать проект
1. Зайди на https://console.firebase.google.com → **Add project** (Создать проект).
2. Назови как угодно (например `gamedag`), Google Analytics можно выключить.

## Шаг 2. Включить Realtime Database
1. В меню слева: **Build → Realtime Database → Create Database**.
2. Локация: любая (например `europe-west1`).
3. Режим запуска: выбери **Start in test mode** (тестовый режим) → Enable.
   - Test mode разрешает чтение/запись всем на 30 дней — для игры с друзьями ок.
   - Позже можно продлить/настроить правила (см. Шаг 5).

## Шаг 3. Получить ключ (web config)
1. Иконка шестерёнки → **Project settings**.
2. Внизу раздел **Your apps** → нажми иконку **</>** (Web).
3. Назови приложение (например `web`) → Register app.
4. Покажется объект `firebaseConfig` — скопируй значения:
   `apiKey`, `authDomain`, `databaseURL`, `projectId`.
   - Если `databaseURL` не показан — возьми его на странице Realtime Database
     (вид `https://gamedag-xxxx-default-rtdb.europe-west1.firebasedatabase.app`).

## Шаг 4. Вставить ключ в игру
Открой файл **`docs/index.html`** в репозитории, найди вверху скрипта блок:

```js
const FIREBASE_CONFIG = {
  apiKey: "PASTE_API_KEY",
  authDomain: "PASTE_PROJECT.firebaseapp.com",
  databaseURL: "PASTE_DATABASE_URL",
  projectId: "PASTE_PROJECT_ID"
};
```

Замени значения `PASTE_...` на свои из Шага 3. Сохрани, закоммить в GitHub —
GitHub Pages обновится через 1–2 минуты. Готово!

> Это **публичный** web-ключ (apiKey), его можно держать в открытом репозитории —
> это нормально для Firebase. Доступ ограничивают правилами базы, а не ключом.

## Шаг 5 (опционально). Правила базы
Test mode истекает через 30 дней. Чтобы продлить — в Realtime Database → **Rules**
вставь простой вариант (для хобби, без авторизации):

```json
{
  "rules": { ".read": true, ".write": true }
}
```

Это открытая база (любой со ссылкой может читать/писать комнаты). Для игры с
друзьями этого достаточно. Если захочешь строже — потом добавим авторизацию.

## Как играть онлайн
1. В боте нажми «🎮 Открыть игры» → выбери игру → **🌐 Онлайн с друзьями**.
2. Введи имя (один раз) → **Создать комнату** → получишь **код из 4 букв**.
3. Друзья открывают бота → та же игра → Онлайн → **Войти по коду** → вводят код.
4. Когда все в лобби, ведущий (создатель комнаты 👑) жмёт **▶️ Начать игру**.

Каждый видит свою роль/слово на своём телефоне, ходы и голосования
синхронизируются в реальном времени.
