# 🌍 Как добавить свою игру в бота

Этот бот — песочница для мини-игр. Любой игрок может предложить свою игру
через **Pull Request** на GitHub. После проверки (модерации) игра появится
во вкладке **«🌍 Игры сообщества»**.

## Как это работает

1. Все игры лежат в папке `docs/games/`.
2. Каждая игра — это **одна папка** со своим файлом `index.html`
   (весь HTML + CSS + JS внутри одного файла).
3. Список игр хранится в файле `docs/games/registry.json`.
4. Бот показывает игры из этого списка и запускает каждую в изолированном
   `<iframe sandbox="allow-scripts">` — у игры **нет доступа** к боту,
   к данным Telegram и к другим играм. Это защищает игроков.

## Шаги для автора

1. Сделай **Fork** репозитория на GitHub.
2. Скопируй файл-шаблон `docs/games/TEMPLATE.html` в новую папку, например:
   `docs/games/moya-igra/index.html`.
   > Имя папки — только латинские буквы, цифры и дефис (например `snake`, `tic-tac-toe`).
3. Напиши свою игру внутри `index.html`. Вся игра должна помещаться в одном файле.
4. Добавь запись о своей игре в `docs/games/registry.json`:

```json
{
  "games": [
    {
      "id": "moya-igra",
      "title": "Моя игра",
      "author": "Твоё имя",
      "icon": "🎮",
      "description": "Короткое описание в одну строку",
      "path": "games/moya-igra/index.html"
    }
  ]
}
```

5. Создай **Pull Request** в основной репозиторий.
6. Дождись проверки. После одобрения и слияния (merge) игра появится в боте.

## Поля в registry.json

| Поле          | Обязательно | Описание                                             |
|---------------|-------------|------------------------------------------------------|
| `id`          | да          | Уникальный идентификатор (латиница, цифры, дефис)    |
| `title`       | да          | Название игры, которое увидят игроки                 |
| `author`      | нет         | Твоё имя или ник                                     |
| `icon`        | нет         | Эмодзи для иконки (по умолчанию 🎮)                  |
| `description` | нет         | Короткое описание в одну строку                      |
| `path`        | да          | Путь к файлу игры относительно папки `docs/`         |

## Правила модерации

Чтобы Pull Request приняли, игра должна:

- ✅ помещаться в одном файле `index.html`;
- ✅ работать на телефоне (адаптивная вёрстка);
- ✅ не содержать рекламы, вредоносного кода и внешних трекеров;
- ✅ не пытаться загружать сторонние скрипты и не запрашивать личные данные;
- ✅ иметь понятное название и описание.

Запрещено: оскорбления, 18+, нарушение авторских прав, майнинг,
попытки «сломать» песочницу.

Спасибо, что делаешь бота лучше! ❤️
